## 填写要点

| 章节 | 要求 |
|------|------|
| 方案选型 | 非平凡处必含最小改动方案；优先推荐最小改动，选复杂方案须说明理由 |
| 组件结构 | 标清复用 vs 新建，新建说明理由 |
| 数据流 | 明确状态归属，避免无依据使用全局状态 |
| 接口契约 | 字段级；区分已有/新增；含参照探查与联调待定清单 |
| 测试策略 | 与 summarized 验收标准对应，供步骤 3 拆任务 |
| 涉及文件 | 精确到路径，作为步骤 4 改动边界 |
| 风险与回滚 | 至少覆盖对现有功能的影响 |
