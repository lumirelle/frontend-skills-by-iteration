## 代码风格

| 维度 | 约定 |
|------|------|
| 命名 | 文件、组件、变量、类型、测试文件 |
| 导入 | 顺序、路径别名、禁止相对路径层级 |
| 组件 | 函数式/选项式、props 命名、文件结构 |
| 状态 | 局部 vs store、何时抽 hook |
| API 调用 | 仅通过 `src/api/` 封装，禁止页面直接 `fetch` |
| 错误处理 | 统一模式（toast / inline / reject） |
| 注释与 TODO | 接口待定用 `TODO(vX.Y.Z): 接口联调待定`（见 `frontend-iteration/references/api-integration-guide.md`） |
