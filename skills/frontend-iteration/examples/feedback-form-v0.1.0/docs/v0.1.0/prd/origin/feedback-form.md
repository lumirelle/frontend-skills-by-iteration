# 用户反馈表单（原始 PRD）

> 这是黄金路径样例的「输入」：一份未经前端归纳的原始 PRD，模拟产品同学给到的内容。

## 背景

希望让已登录用户能在「我的」页面提交反馈，收集产品改进意见。当前没有任何反馈入口。

## 需求

- 在「我的」页面增加一个「意见反馈」入口，点击进入反馈表单。
- 反馈表单包含：
  - 反馈类型（下拉选择：功能建议 / 问题反馈 / 其他）
  - 反馈内容（多行文本）
- 用户填写后点击「提交」，提交成功后给出提示并清空表单。
- 反馈内容是必填的，不能为空。
- 只有登录用户能提交。

## 其他

- 后端会提供一个提交接口，具体字段待定。
- 希望尽快上线，先做基础版本即可，后续再考虑图片上传、历史记录等。
